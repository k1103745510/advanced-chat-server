개인정보 처리방침

본 API 서비스는 OpenAI API를 기반으로 동작하며, 사용자의 개인정보 보호를 최우선으로 생각합니다. 이 문서는 본 서비스가 어떤 방식으로 데이터를 처리하고 저장하는지를 설명합니다.

1. 수집하는 정보

본 서비스는 사용자의 이름(`client_name`)과 질의 입력(`input`)만을 수집하며, 사용자가 직접 요청한 대화와 관련된 내용만을 처리합니다.

2. 수집된 정보의 이용 목적

입력된 데이터는 OpenAI의 GPT 모델과의 대화 생성을 위해서만 사용됩니다. 별도의 마케팅, 광고, 타 서비스 공유 목적 등에는 일절 사용되지 않습니다.

3. 데이터 저장

  • 대화 기록은 서버 내 파일(`conversation_histories/{client_name}_history.json`)로 저장되며, 향후 대화 컨텍스트를 유지하는 용도로만 사용됩니다.
  • 사용자의 대화 기록은 해당 사용자가 직접 요청(DELETE `/history`) 시 즉시 삭제됩니다.
  • 모델 설정 정보는 `configs/{client_name}_config.json` 에 저장되며, 특정 사용자 설정 유지 목적으로만 사용됩니다.

4. 외부 전송 여부

입력된 대화 내용은 OpenAI의 GPT API (https://api.openai.com)로 전달됩니다. 이 외에 제3자 서비스, 분석 도구, 광고 네트워크 등으로 전송되지 않으며, 별도의 추적 기능 또한 포함하지 않습니다.

5. 데이터 보존 기간

대화 기록은 사용자가 직접 삭제하지 않는 한 계속 보존됩니다. 단, 사용자가 삭제를 요청하면 즉시 삭제되며 복구되지 않습니다.

6. 사용자의 권리

사용자는 다음의 권리를 보장받습니다:

  • 자신의 대화 기록을 조회할 수 있습니다 (`GET /history`).
  • 자신의 대화 기록을 언제든지 삭제할 수 있습니다 (`DELETE /history`).

7. 보안

서버는 사용자 데이터가 외부에 노출되지 않도록 하기 위해 디렉토리 기반으로 구분 저장되며, 외부 접속으로부터 보호됩니다. 다만, 사용자는 자신의 클라이언트 이름(`client_name`)이 유출되지 않도록 주의해야 합니다.
